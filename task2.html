Combinator Selector and Attribute Selector

Combinator Selector:

These selector used to specify the relation between two selectors.
There are four types of combinator selectors.

  • Descendant Selector:
    The descendant selector matches all elements that are descendants of a specified element.

    Example:

                    div p{
                    color:blue;
                }
  • Child Selector:
    The child selector selects all elements that are the children of a specified element.

    Example:

                    div>p{
                    color:blue;
                }
  • Adjacent Sibling Selector:
    This selector used to select an element that is directly after a specific element.
    Example:

     
                    div+p{
                        background-color:green;
                    }
  • General Sibling Selector:
    This selector selects all the elements that are siblings of a specified element
    Example:

                    h1~p{
                    ------;
                }

Attribute Selector:

These selector is used to select elements with a specified attribute.

Example:

            p[id]{
                background-color:green;
            }

CSS[attribute="value"]Selector:

These selector is used to select elements with a specified attribute and value.

Example:

       p[id="idname"]{
        color:blue;
       }

CSS[attribute~="value"]Selector:

These Selector is used to select elements with an attribute value containing specific word.

Example:

    p[id~="para"]{
        color:blue;
    }

CSS[attribute|="value"]Selector:

These selector is used to select elements with a specified attribute,whose value can be exactly specified value or specfied value followed by hyphen(-).

Example:

    p[id|="para"]{
        background-color=green;
    }

CSS[attribute$="value"]Selector:

These selector used to select elements whose attributes value ends with a specified value.

Example:

    p[id$="para"]{
        text-align:center;
    }

CSS[attribute*="value"]Selector:

These selector used to select elements whose attribute value contains a specified value.

Example:

    p[id*="first"]{
        color:blue;
    }
