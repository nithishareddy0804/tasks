Meta Tag:

The meta tag defines metadata about an HTML document. Metadata is data (information) about data. Meta tags always go inside the head element.

Meta Attributes:

  • charset: Specifies the character encoding for the HTML document.
    • Syntax:< meta charset="UTF-8">

  • Content: Specifies the value associated with the http-equiv or name attribute.
    • Syntax:< meta content="">

  • http-equiv: Provides an HTTP header for the information/value of the content attribute.
    • Syntax:< meta http-equiv="">

  • name:Specifies a name for the metadata
    • Syntax:< meta name="">

Examples:

  • Define keywords for search engines:
    • < meta name="keywords" content="HTML,CSS,JS">

  • Define a description of your web page:
    • < meta name="description" content="Free web tutorials for html and css">

  • Define the author of a page:
    • < meta name="author" content="Nithisha">

  • Refresh document every 30 seconds:
    • < meta http-equiv="refresh" content="5">

  • Setting the viewport to make your website look good on all devices:
    • < meta name="viewport" content="width=device-width, initial-scale=1.0">
