CSS (Cascading Stylesheets)

CSS:

  • Cascading Style Sheets is used to add styles to HTML document.
  • CSS is an important way to control how our web pages look.

CSS Styles in three ways:

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS:

  • Inline Css is a method of applying styles directly within HTML element using style attribute
    • Example: <p style="color: aliceblue;>"hello</p>

Internal CSS:

  • The internal css is a set of styles created as part of HTML document.

External CSS:

  • External Css is used to style multiple HTML documents with a single style sheet.
